═══════════════════════════════════════════════════════════════
  FILE DESCRIPTIONS
═══════════════════════════════════════════════════════════════

mysql/
  → Complete portable MySQL 8.0.44 installation
  → Extract MySQL ZIP contents here
  → Must include bin/mysqld.exe

app/
  → All FinalPOS application files
  → Copy from FinalPOS/bin/Release/
  → Include ALL DLLs and resources

config/my.ini
  → MySQL server configuration
  → Port: 3307
  → Password: Shivaanica
  → Database: POS_NEXA_ERP

scripts/init.sql
  → Sets root password
  → Creates POS_NEXA_ERP database
  → Grants privileges

scripts/schema.sql
  → Your complete database schema
  → Will be imported during installation
  → Replace placeholder with actual schema

scripts/Start-MySQL.bat
  → Starts MySQL server on port 3307
  → Checks if already running
  → Verifies port listening

scripts/Stop-MySQL.bat
  → Safely stops MySQL server
  → Graceful shutdown first
  → Force kill if needed

scripts/Check-MySQL.bat
  → Checks MySQL process status
  → Verifies port 3307 listening
  → Tests database connection
  → Shows MySQL version

scripts/Install-MySQL.bat
  → Initializes MySQL data directory
  → Starts MySQL server
  → Runs init.sql
  → Imports schema.sql
  → Called automatically during install

InnoSetupScript.iss
  → Complete Inno Setup installer script
  → Handles all installation steps
  → Creates shortcuts
  → Manages uninstallation

README.md
  → Complete documentation
  → Installation instructions
  → Troubleshooting guide
  → Configuration details

QUICK-START.txt
  → Quick reference guide
  → Essential commands
  → Common issues and solutions
